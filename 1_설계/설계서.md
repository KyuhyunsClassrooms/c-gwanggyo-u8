# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
컴퓨터와하는 가위바위보 게임
-컴퓨터가 가위 바위 보 중 하나를 고른다
-사람도 가위바위보중 하나를 고른다.
-사람이 이기면 '당신이 이겼습니다.' 출력
-사람이 지면 '당신이 졌어요.' 츨력
-연속 승리시 다른 멘트 출력
-연속 패배시 다른 멘트 출력

## 2. 핵심 기능 목록
(구현할 기능들을 순서대로 나열하세요.)
1.  rand()%3 +1을 이용하여 가위바위보 랜덤 선택 기능 넣기
2.  ssrand(time(NULL)) 넣어서 랜덤으로 잘 나오게 하기
3.  if 조건문을 이용해 가위바위보 이기고 지는 상황 만들기
4.  while문을 이용하여 잘 못 입력 받는 경우 처리하기
