## 🐛 발견한 오류 #1

### 1) 어떤 문제가 발생했나요?
(예: 프로그램을 실행하면 이상한 값이 나왔어요 / 컴파일이 안 돼요)

```
에러가 난다.
```

### 2) 오류가 발생한 코드
```c
printf("당신이 이겼어요!"\n);
```

### 3) 왜 오류가 생겼나요?


```
\n을 "" 안쪽에 적어야 하는데 밖에 적었다.
```

### 4) 어떻게 고쳤나요?
```c
printf("당신이 이겼어요!\n");
```

이번 프로젝트에서 오류를 고치면서 새롭게 알게 된 점이나 느낀 점을 자유롭게 적어보세요.

```

```
